# sandy-plugins

A curated collection of plugins for [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Plugins

| Plugin | Description |
|--------|-------------|
| [synthkit](https://github.com/rappdw/synthkit) | Document synthesis toolkit — guided exploration, markdown to PDF/DOCX/HTML/email |
| [ait](https://github.com/rappdw/ait) | Personal AI team and knowledge base — specialized agents, dynamic hiring, and searchable information repository |

## Installation

```bash
claude plugin install sandy-plugins
```
